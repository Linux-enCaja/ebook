\subsection[Motivación]{Ventajas al Usar Grillas Inteligentes}
\begin{frame}
  \frametitle{Ventajas al Usar Grillas Inteligentes}
      \begin{itemize}
        \item Ahorro de energía.
        \item Mejoramiento de la calidad del servicios.
        \item Reducción de los costos operativos.
          \begin{itemize}
            \item Eliminación del costo de la lectura.
            \item Rápida detección de cortes/fallas
            \item Conexión/desconexión Remota	
          \end{itemize} 
        \item Disminución de las pérdidas de energía.
          \begin{itemize}
            \item Permite detectar sobre-dimensionamiento de transformadores.
            \item Facilita la detección de fraudes.
          \end{itemize} 
        \item Permite manejar precios dinámicos.
        \item Permite el control de la carga.
        \item Conocimiento del consumo en tiempo real (usuarios).
      \end{itemize}
\end{frame}


\subsection[Motivación]{Justificación}
\begin{frame}
  \frametitle{Justificación}
    \begin{block}{}
      \begin{itemize}
       \item En la actualidad existen muchas opciones que permiten realizar diferentes mediciones sobre el sistema de distribución de energía eléctrica.
       \item No se ofrece una solución completa que incluya los diferentes niveles del sistema.
       \item Las soluciones actuales las proveen diferentes fabricantes que utilizan diferentes tecnologías, requiriendo un gran esfuerzo de integración para poder recolectar y procesar la información.
       \item Estos sistemas son diseñados para países con situaciones políticas, económicas, sociales diferentes a las de los países latinoamericanos.
      \end{itemize}
    \end{block}
\end{frame}


\section{Solución}
\begin{frame}
  \frametitle{Solución}
   \only<1>{
     \begin{block}{}
       \begin{itemize}
        \footnotesize
        \item Crear equipos diseñados a la medida y acorde a las necesidades y requerimientos del sector eléctrico nacional.
          \begin{itemize}
            \item Apoyo a la industria electrónica nacional.
            \item Elimina dependencia tecnológica.
            \item Desarrollo tecnológico del país.
          \end{itemize}
        \item Implementar redes de transmisión inteligente, robustas ante condiciones de falla. 
        \item Medición del consumo de energía, corte y reconexión de forma remota. 
       \end{itemize}
     \end{block}
   }
   \only<2>{ 
      \begin{block}{}
        Registro en tiempo real de la calidad del suministro eléctrico y del usuario final. 
          \begin{itemize}
          \footnotesize
	    \item Monitoreo de el sistema de iluminación.
	    \item Detección temprana de interrupciones del servicio.
	    \item Perfiles de carga a través del día.
	    \item Perfiles de corriente por fase.
	    \item Detección de picos de demanda. 
	    \item Balances de energías activa, reactiva y aparente por fase.
	    \item Generación de alarmas por manipulación mal-intencionada de los equipos de medida. 
          \end{itemize}
      \end{block}
   }

\end{frame}

\subsection[Sistema de Medición Propuesto]{Macromedidor}
\frame[c]
{
  \frametitle{Características del Macromedidor}

      \begin{columns}[c]
        \begin{column}{0.45\textwidth}
          \pgfimage[height=5cm]{./img/comunication2}
        \end{column}
        
	\begin{column}{0.55\textwidth}
         
         \only<1>{
          \begin{block}{\small Medición}
           \tiny
            \begin{itemize}
	      \item Medición trifásica de energía.
	      \item Medición de corriente del neutro.
	      \item Medición de energía en la línea de alumbrado público.
	      \item Calculo de energía y potencia (activa, reactiva y aparente).
	      \item Calculo de Balance, Factor K, Análisis de armónicos.
	      \item Cálculo de flicker, Cálculo de eventos (SAGs y SWELLs)
	      \item Registro de DES y FES, Capacidad de almacenamiento.
	      \item Baterías de Ion de Litio.
             \end{itemize}
          \end{block}

            \begin{alertblock}{\tiny normas}
             \tiny
              \begin{itemize}
                \item Análisis de armónicos (IEC 61000-4-7 y IEEE 519)
                \item Calidad de energía y potencia  (IEC 61000-4-30 clase A, IEEE1159, IEEE1459)
	        \item Análisis de armónicos (IEC 61000-4-7 y IEEE 519)
	        \item Flickers (IEC 61000-4-15)
	        \item SAGs y SWELLs (61000-4-30 clase A)
	        \item DES y FES GREG 083 y 091 de 2004
               \end{itemize}
            \end{alertblock}
         }
          \only<2>{
          \begin{block}{\small Ventajas y Beneficios}
           \tiny
            \begin{itemize}
	      \item Posibilidad de detectar maniobras y cortes del servicio.
	      \item Cálculo de la carga del transformador, lo cual permite su correcto dimensionamiento.
	      \item La información recolectada de forma periódica proporciona información precisa de la carga, lo cual facilita los procesos de programación del suministro, operaciones de conmutación, planificación, balance, etc.
             \end{itemize}
          \end{block}

             \begin{block}{\small Costos (estimados)}
             \tiny

               100    & 670\\

               1000   & 585\\
               \hline 
              \end{tabular}
             \end{center}
            \end{block}

	      \item Cada poste posee medidores, relés electrónicos y acometidas para cada usuario.

	      \item Balance de Carga en el transformador.
	      \item Comunicación desde el transformador y el centro de control vía WiMax.
	      \item Permite desconexión individual.
