\subsection{SIE: Plataforma hardware copyleft }
 
 \begin{frame}

  \frametitle{SIE: Plataforma hardware copyleft}

     \begin{block}{ }
      \begin{itemize}
       \item Acceso a los archivos de diseño realizados en herramientas abiertas (kicad en este caso)
       \item Posibilidad de hacer cambios físicos y funcionales.
       \item Acceso a notas de aplicación, diseños de referencia.
       \item Acceso al software básico: \textit{bootloader} \textit{kernel} \textit{sistema de archivos}
       \item Soporte a través de listas de discusión.
       \item Acceso a los fabricantes de las tarjetas.
      \end{itemize}
     \end{block}

    \begin{alertblock}{}
      Todo esto puede descargarse de: http://wiki.linuxencaja.net/wiki/GIT

}
       \begin{block}{}
         \begin{itemize}
          \item Encuentran la metodología adecuada para generar habilidades necesarias para el desarrollo de aplicaciones comerciales
          \item Se proporcionan las herramientas necesarias para lograr el objetivo final.
          \item Son conscientes de que se busca que ellos generen productos novedosos y de esta forma generar empleo.
          \item Se muestra que los estudiantes no tienen ningún problema en utilizar herramientas licenciadas de forma ilegal.
          \item Prefieren el uso de las herramientas libres aunque su utilización sea mas compleja.
         \end{itemize}
       \end{block}

\begin{frame}
  \frametitle{Conclusiones}
       \begin{alertblock}{}
         \begin{itemize}
          \item El hardware copyleft es una herramienta poderosa para la creación de habilidades necesarias para concebir, diseñar, implementar y operar sistemas digitales
          \item Las actividades propuestas en las tres asignaturas del área tienen como objetivo generar en el estudiante las habilidades necesarias que le permitan diseñar sistemas digitales con grado de complejidad creciente, hasta llegar a un sistema que puede ser comercializable y satisface una necesidad de una determinada comunida
          \item La utilización de herramientas de bajo nivel permite que el estudiante conozca y controle los diferentes pasos de la metodología de diseño y sea capaz de ajustarlas para diferentes situaciones, esto hace que se adquiera un conocimiento sobre la tecnología sin crear dependencia
         \end{itemize}
       \end{alertblock} 
\end{frame}
